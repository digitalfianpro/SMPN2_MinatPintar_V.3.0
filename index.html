Panduan Operasional & Bantuan Cloud

1. Alur Persiapan & Pengaturan BK

Sebelum menyebarkan tes ke siswa, Guru BK wajib mengatur data identitas sekolah pada menu Data Sekolah. Data ini akan otomatis dicetak pada Kop Laporan Excel beserta format tanda tangan Kepala Sekolah dan Guru BK:

  • Buka menu Data Sekolah di sidebar.
  • Masukkan nama sekolah resmi (misal: SMP Negeri 2 Manokwari).
  • Isi identitas kepala sekolah, Guru BK, NIP, serta kota penetapan dokumen laporan.
  • Tekan tombol Simpan Data Sekolah. Data akan langsung terunggah ke Cloud Firestore.

2. Mengaktifkan Instrumen Tes Siswa

Secara bawaan, semua instrumen tes dalam kondisi terkunci (ditutup). Siswa tidak akan bisa melihat instrumen tes sebelum Guru BK membukanya secara resmi dari Dashboard:

  • Buka menu Bank Soal CBT pada bilah menu kiri.
  • Bapak akan melihat daftar 5 instrumen tes terstandar (Kepribadian, Penjurusan, Kemampuan Spasial, Gaya Belajar VAK, Bakat & Minat).
  • Geser sakelar/switch di sudut kanan bawah setiap kartu tes. Warna akan berubah menjadi Hijau Menyala (Active/Terbuka).
  • Siswa yang memuat ulang halaman tes mereka sekarang sudah bisa langsung melihat tes tersebut dan mulai mengerjakannya.

3. Menyebarkan Link CBT kepada Siswa

Setelah tes aktif, Guru BK tinggal membagikan alamat akses tes. Di dalam menu Bank Soal CBT, terdapat kotak tautan otomatis yang telah terkalibrasi khusus:

  • Pada bagian atas menu Bank Soal, cari kolom Link Siswa (Cloud). Tautan simpel otomatis (Berakhiran `#smpn2`) sudah dibuat di sana.
  • Klik tombol Salin Link di sebelahnya. Alamat website tes akan langsung tersalin ke clipboard laptop Bapak.
  • Bagikan link tersebut ke grup WhatsApp kelas, proyeksikan di papan tulis kelas, atau ketikkan di komputer laboratorium sekolah.
  • Siswa cukup membuka link tersebut melalui HP masing-masing atau komputer lab untuk langsung masuk ke Portal Tes.

4. Monitoring Peserta Live

Selama tes berlangsung, Bapak tidak perlu berkeliling ruang kelas untuk mengecek progres siswa secara manual. Pantau semuanya secara real-time dari meja pengawas Bapak:

  • Pindah ke menu Peserta Live.
  • Setiap kali ada siswa yang mengisi formulir nama dan menekan tombol mulai, nama mereka akan langsung muncul di tabel dengan status Mengerjakan (Kuning Berkedip).
  • Jika waktu habis atau siswa mengirim jawaban, status mereka akan otomatis berganti secara instan menjadi Selesai (Hijau).
  • Jika ada siswa yang lupa menekan tombol selesai padahal mereka sudah meninggalkan kelas, Bapak dapat menekan tombol merah Akhiri Sesi untuk mematikan dan menyimpan jawaban mereka secara paksa.

5. Mengunduh Rekapan Analisis & Ekspor Excel

Setelah tes selesai seluruhnya, Bapak dapat mengunduh rekapitulasi data hasil tes psikologis siswa beserta hasil interpretasi saran bimbingan konseling otomatis:

  • Buka menu Rekapan Analisis.
  • Periksa seluruh baris nilai, skor tes, dan prediksi saran dari sistem kecerdasan buatan bimbingan konseling bapak.
  • Gunakan fitur Edit untuk mengkoreksi/menambahkan catatan pada hasil analisis AI jika diperlukan.
  • Tekan tombol hijau Export XLS di bagian kanan atas. File laporan `.xls` terstandar akan langsung terunduh secara instan ke folder unduhan laptop Bapak.
  • File laporan tersebut sudah rapi, ramah cetak, dilengkapi Kop Surat sekolah yang Bapak atur tadi, serta sudah memiliki kolom tanda tangan resmi Guru BK dan Kepala Sekolah di bagian bawah tabel.
  • Gunakan tombol Backup DB secara berkala untuk mengamankan salinan data siswa ke hardisk lokal Anda sebagai arsip cadangan mandiri.
